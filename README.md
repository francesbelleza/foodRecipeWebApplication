# foodRecipeWebApplication
The objective of this project is to create a Food Recipe web application
that allows users to create, view, update and delete food recipes.
 ~ project for CMPE131 ~

### **Tech Stack**
 - Python/Flask
 - SQLAlchemy
 - FLask-WTF
 - Flask-Login
